# 紫微斗数星盘 · 宫位卡片布局设计规范

> 本文档记录 iztro 前端星盘页面中每个宫位卡片的布局设计，
> 是后续开发或重构的参考标准。

---

## 1. 整体布局：4×4 星盘

星盘采用 **4×4 CSS Grid** 布局，12 个宫位环绕排列，中心 4 格留空。

```
┌─────┬─────┬─────┬─────┐
│ 巳  │ 午  │ 未  │ 申  │
│ (3) │ (4) │ (5) │ (6) │
├─────┼─────┼─────┼─────┤
│ 辰  │     │     │ 酉  │
│ (2) │ 空  │ 空  │ (7) │
├─────┼─────┼─────┼─────┤
│ 卯  │     │     │ 戌  │
│ (1) │ 空  │ 空  │ (8) │
├─────┼─────┼─────┼─────┤
│ 寅  │ 丑  │ 子  │ 亥  │
│ (0) │ (11)│ (10)│ (9) │
└─────┴─────┴─────┴─────┘
```

**关键参数：**
- Grid 容器：`grid-template-columns: repeat(4, 1fr)`，`gap: 0`
- 宫位卡片：`outline: 1px solid #bbb`（不占布局空间）
- 中心空格：`outline: none`，完全透明
- 容器比例：`aspect-ratio: 1`（正方形）
- 宫位索引 0~11 对应地支 **寅卯辰巳午未申酉戌亥子丑**

## 2. 宫位卡片内部布局

每个宫位卡片自身是一个 **3 列 × 4 行的 CSS Grid**：

```
grid-template-rows:    1fr auto auto auto
grid-template-columns: 1fr 1fr 1fr
```

### 2.1 区域划分

```
┌─────────────────────────────────────┐
│  Row 1 (col 1-3)  星曜区 (stars)    │  ← 1fr
├─────────────────────────────────────┤
│  Row 2 (col 1-3)  大限区 (decadal)   │  ← auto
├───────────────────┬─────────────────┤
│  Row 3 (col 1)   │  Row 3 (col 2-3) │  ← auto
│  神煞区 (deities) │  空(留白)        │
├───────────────────┼──────┬──────────┤
│  Row 4 (col 1)   │ col2 │  col 3    │  ← auto
│  神煞区(续)       │ 宫名  │ 干支区   │
└───────────────────┴──────┴──────────┘
```

### 2.2 各区域详细说明

#### 2.2.1 星曜区 (stars-area)
- **位置**：第 1 行，跨 3 列
- **布局**：`flex-wrap: wrap`，从左到右排列
- **高度**：`1fr`（占据顶部剩余空间）
- **内容**：星曜按 **主星 → 辅星 → 煞星 → 杂曜** 顺序排列

每个星曜为一个 `star-item`（flex column）：
```
┌─────┐
│ 破  │  ← star-name (writing-mode: vertical-lr)
│ 军  │     竖排文字，红(主)/绿(辅)/橙(煞)/灰(杂)
├─────┤
│ 旺  │  ← brightness (8px, 灰色)
├─────┤
│ 禄  │  ← mutagen (7px, 红底白字标签)
└─────┘
```

- 星名：`writing-mode: vertical-lr` 竖排排列
- 亮度：紧挨在星名正下方
- 四化：紧挨在亮度下方，红底白字小标签

#### 2.2.2 流耀区 (transit-area) — 绝对定位
- **位置**：absolute 定位，卡片右侧垂直居中
- **布局**：`flex-direction: row-reverse`，从右到左排列
- **顺序**：大限 → 流年 → 流月 → 流日 → 流时（从右到左）
- **样式**：每个星曜单独显示，`writing-mode: vertical-lr` 竖排
- **颜色**：
  - 大限(decadal)：橙色 `#c55a11`
  - 流年(yearly)：蓝色 `#2e75b6`
  - 流月(monthly)：绿色 `#548235`
  - 流日(daily)：金色 `#bf8f00`
  - 流时(hourly)：棕色 `#843c0c`
- **触发**：仅当计算了运限数据后显示
- **隐藏身宫**：当流耀区显示时，身宫自动隐藏

#### 2.2.3 大限区 (decadal-area)
- **位置**：第 2 行（原第 3 行），跨 3 列
- **对齐**：`text-align: center`
- **内容**：大限起止年龄，如 `2-11岁`
- **装饰**：上下虚线边框 `border-top/bottom: 1px dotted #ddd`

#### 2.2.4 神煞区 (deities-area)
- **位置**：第 3 ~ 4 行，第 1 列
- **布局**：flex column，从上到下排列
- **对齐**：`align-self: end`（底部对齐）
- **内容**：博士十二神、将前十二神、岁前十二神
- **顺序**：博士(浅绿色 `#66bb6a`) → 将前 → 岁前（从上到下）
- **注意**：长生十二神不在此处显示（移至右下角）

#### 2.2.5 宫名区 (name-area)
- **位置**：第 4 行，第 2 列
- **对齐**：底部居中
- **样式**：红色 `#c0392b`，13px，粗体

#### 2.2.6 干支区 (stem-area)
- **位置**：第 4 行，第 3 列
- **对齐**：右下角紧贴边框（`align-self: end; margin: 0 -3px -3px 0`）
- **内容**：
  - 上方：长生十二神（8px，灰色，竖排 `writing-mode: vertical-lr`）
  - 下方：天干地支（11px，深灰色，竖排 `writing-mode: vertical-lr`）

#### 2.2.7 身宫标记 (body-mark) — 绝对定位
- **位置**：absolute 定位，卡片右侧垂直居中
- **样式**：白底红字红边框圆角 `border-radius: 4px`
- **文字**：竖排 `writing-mode: vertical-lr`，"身宫"
- **隐藏条件**：当计算了运限数据时隐藏（与流耀区互斥）

## 3. 视觉层次总结

```
┌──────────────────────────────────────┐
│ 破  左  文  擎  天  天               │  ← 顶部：星曜区（主→辅→煞→杂）
│ 军  辅  昌  羊  巫  哭   运 流 流     │     右侧中间：流耀区（从右到左）
│ 旺  庙  庙  陷  旺  平   魁 魁 禄     │     垂直居中，颜色区分 scope
│ 禄                      (橙蓝绿金棕)  │
├──────────────────────────────────────┤
│             2-11岁                   │  ← 大限区（居中）
├──────────────────────────────────────┤
│ 博士                              │
│ 将前       命宫                     │  ← 底部：神煞(左) 宫名(中)
│ 岁前                     长生十二神  │        干支(右下)
│                          丙寅       │
└──────────────────────────────────────┘
```

---

## 4. 响应式行为

- 星盘容器：`max-width: 860px`，`aspect-ratio: 1` 保持正方形
- 星曜区：`flex-wrap: wrap`，星曜过多时自动换行
- 字体大小：星名 10px，亮度 8px，四化 7px，神煞 8px，大限 9px，干支 11px
