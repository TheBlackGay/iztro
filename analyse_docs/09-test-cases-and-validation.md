# 测试用例集与输入验证规范

> 本文档提供语言无关的测试用例和输入验证规则，
> 供其他语言实现时用于验证排盘结果的正确性。

---

## 1. 输入验证规范

### 1.1 日期验证

| 规则 | 说明 |
|------|------|
| 日期格式 | `YYYY-M-D`（如 `2000-8-16`），不支持前导零 |
| 支持年份范围 | 1900~2100（依赖农历数据表范围） |
| 农历日期 | 有对应的阳历日期即可 |
| 闰月 | 农历闰月用 `isLeapMonth` 标记 |

### 1.2 时辰验证

| timeIndex | 有效范围 | 说明 |
|-----------|---------|------|
| 0 | 早子时 | 00:00~01:00 |
| 1~11 | 丑~亥时 | 正常时辰 |
| 12 | 晚子时 | 23:00~00:00 |
| 其他 | **无效** | 应返回错误 |

### 1.3 性别验证

| 输入值（中文） | 输入值（英文） | 阴阳 |
|--------------|--------------|------|
| 男 | male | 阳 |
| 女 | female | 阴 |
| 其他 | 其他 | **无效** |

### 1.4 配置项验证

| 配置项 | 合法值 | 默认值 |
|--------|-------|-------|
| `yearDivide` | `normal`, `exact` | `normal` |
| `horoscopeDivide` | `normal`, `exact` | `normal` |
| `ageDivide` | `normal`, `birthday` | `normal` |
| `dayDivide` | `current`, `forward` | `current` |
| `algorithm` | `default`, `zhongzhou` | `default` |
| `fixLeap` | `true`, `false` | `true` |
| `astroType` | `heaven`, `earth`, `human` | `heaven` |
| `language` | `zh-CN`, `zh-TW`, `en-US`, `ja-JP`, `ko-KR`, `vi-VN` | `zh-CN` |

```
仆役   selfMutagedOneOf(['科','权']) → true

命宫.mutagedPlaces() → [命宫, 迁移, 仆役, 兄弟]
```
