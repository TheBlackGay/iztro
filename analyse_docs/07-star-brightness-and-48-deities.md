## 2. 四十八神煞

### 2.1 长生十二神

**12 神顺序**（固定，不可变）：
```
长生 → 沐浴 → 冠带 → 临官 → 帝旺 → 衰 → 病 → 死 → 墓 → 绝 → 胎 → 养
```

**內部的 key 对应**：
```
changsheng, muyu, guandai, linguan, diwang, shuai, bing, si, mu, jue, tai, yang
```

**起始位置按五行局**：

| 五行局 | 起始地支 |
|-------|---------|
| 水二局 (water2nd) | 申 (shen) |
| 木三局 (wood3rd) | 亥 (hai) |
| 金四局 (metal4th) | 巳 (si) |
| 土五局 (earth5th) | 申 (shen) |
| 火六局 (fire6th) | 寅 (yin) |

**方向判断**：
```
isForward = (性别为阳 && 年支为阳) || (性别为阴 && 年支为阴)
// true → 顺时针 (idx = (startIdx + i) mod 12)
// false → 逆时针 (idx = (startIdx - i) mod 12)
```

### 2.2 博士十二神

**12 神顺序**（固定）：
```
博士 → 力士 → 青龙 → 小耗 → 将军 → 奏书 → 飞廉 → 喜神 → 病符 → 大耗 → 伏兵 → 官府
```

**內部 key 对应**：
```
boshi, lishi, qinglong, xiaohao, jiangjun, zhoushu, faylian, xishen, bingfu, dahao, fubing, guanfu
```

**起始位置**：禄存所在宫位

**方向**：同长生十二神判断规则

### 2.3 岁前十二神

**12 神顺序**：
```
岁建 → 晦气 → 丧门 → 贯索 → 官符 → 小耗 → 大耗(全书)/岁破(中州) → 龙德 → 白虎 → 天德 → 吊客 → 病符
```

**內部 key 对应**：
```
suijian, huiqi, sangmen, guansuo, gwanfu, xiaohao, dahao(全书)/suipo(中州), longde, baihu, tiande, diaoke, bingfu
```

**起始位置**：年支所在宫位

**方向**：顺时针逐宫排列

### 2.4 将前十二神

**12 神顺序**：
```
将星 → 攀鞍 → 岁驿 → 息神 → 华盖 → 劫煞 → 灾煞 → 天煞 → 指背 → 咸池 → 月煞 → 亡神
```

**內部 key 对应**：
```
jiangxing, panan, suiyi, xiishen, huagai, jiesha, zhaisha, tiansha, zhibei, xianchi, yuesha, wangshen
```

**将星起始位置按年支三合**：

| 年支三合 | 将星起始地支 |
|---------|-----------|
| 寅/午/戌 | 午 (woo) |

| 巳/酉/丑 | 酉 (you) |
