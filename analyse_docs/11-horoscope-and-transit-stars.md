# 流耀运限计算规范

> 本文档提供运限和流耀的完整计算规则，
> 供其他语言实现时参考。

---

## 1. 运限系统概述

紫微斗数的运限系统分为五个层次：

| 层次 | 英文 scope | 时间跨度 | 计算基准 |
|------|-----------|---------|---------|
| 大限 | decadal | 10 年 | 命宫起，五行局定起运年龄 |
| 小限 | age | 1 年 | 年支起，逐年移一宫 |
| 流年 | yearly | 1 年 | 流年地支为命宫 |
| 流月 | monthly | 1 月 | 流年命宫起，正月起 |
| 流日 | daily | 1 日 | 流月命宫起 |
| 流时 | hourly | 1 时辰 | 流日命宫起 |

---

## 2. 大限计算

### 2.1 算法

```
大限方向：
  isForward = (性别为阳 && 年支为阳) || (性别为阴 && 年支为阴)
  // true → 从命宫顺时针, false → 从命宫逆时针

大限起始年龄：
  startAge = 五行局数值 + 10 * i   (i = 0, 1, ..., 11)

大限天干：
  startStem = 五虎遁[年干]     // 寅月的天干
  stem = (startStem + palaceIdx) mod 10

大限地支：
  branch = (palaceIdx + 2) mod 12
```

### 2.2 示例

五行局=木三局(3)，阳男顺行：

| 大限序号 | 宫位偏移 | 宫位名 | 起始年龄 | 结束年龄 |
|---------|---------|--------|---------|---------|
| 0 | +0 | 命宫 | 3 | 12 |
| 1 | +1 | 父母宫 | 13 | 22 |
| 2 | +2 | 福德宫 | 23 | 32 |
| 3 | +3 | 田宅宫 | 33 | 42 |
| 4 | +4 | 官禄宫 | 43 | 52 |
| 5 | +5 | 交友宫 | 53 | 62 |
| 6 | +6 | 迁移宫 | 63 | 72 |
| 7 | +7 | 疾厄宫 | 73 | 82 |
| 8 | +8 | 财帛宫 | 83 | 92 |
| 9 | +9 | 子女宫 | 93 | 102 |
| 10 | +10 | 夫妻宫 | 103 | 112 |
| 11 | +11 | 兄弟宫 | 113 | 122 |

### 2.3 大限四化

大限的天干也产生四化（运限四化），用该大限的天干查四化表。

```
大限四化 = 四化表[大限天干]
```

## 3. 小限计算

### 3.1 算法

**口诀**：
```
寅午戌人辰上起，申子辰人自戌宫
巳酉丑人未宫始，亥卯未人起丑宫
```

**起始宫位**：

| 年支 | 小限起始宫位索引 |
|------|---------------|
| 寅/午/戌 | 辰(2) |
| 申/子/辰 | 戌(8) |
| 巳/酉/丑 | 未(5) |
| 亥/卯/未 | 丑(11) |

**方向**：同大限（阳男阴女顺行，阴男阳女逆行）

**年龄数组**：每宫 10 个年龄，步长 12

```
宫位 i 的小限年龄 = [12*j + (i+1) for j in 0..9]
```

例如，从辰宫（索引=2）起始，阳男顺行：

| 宫位索引 | 宫名 | 小限年龄 |
|---------|------|---------|
| 2 | 辰 | 1, 13, 25, 37, 49, 61, 73, 85, 97, 109 |
| 3 | 巳 | 2, 14, 26, 38, 50, 62, 74, 86, 98, 110 |
| 4 | 午 | 3, 15, 27, 39, 51, 63, 75, 87, 99, 111 |
| ... | ... | ... |

### 3.2 小限四化

小限不产生四化。

## 4. 流年计算

### 4.1 流年命宫

```
流年命宫 = 流年地支所在的宫位索引
```

例如：流年为 2025 年乙巳年，地支为巳(索引 5)，则：

```
流年命宫 = palaceIndex(巳) = 5 - 2 = 3 (宫位索引)
```

对应的宫位名称 = PALACES[(3 - soulIndex) mod 12]

### 4.2 流年天干地支

```
yearIndex = 目标流年年份
stem = (yearIndex - 4) mod 10
branch = (yearIndex - 4) mod 12
```

### 4.3 流年四化

流年天干也产生四化（流年四化），使用流年的天干查四化表。

```
流年四化 = 四化表[流年天干]
```

---

## 5. 流月计算

### 5.1 流月命宫

```
流月命宫 = (流年命宫 + (目标月 - 1)) mod 12
```

例如：流年命宫在 3(巳)，目标月为 5 月：
```
流月命宫 = (3 + 5 - 1) mod 12 = 7 (酉)
```

### 5.2 流月天干地支

```
monthStem = (流年天干 × 2 + 流月月份) 的某种变换
实际使用五虎遁：startStem = 五虎遁[流年天干], monthStem = (startStem + 流月index) mod 10
monthBranch = (流月月份 - 1 + 2) mod 12   // 寅月=0
```

---

## 6. 流日计算

### 6.1 流日命宫

```
日偏移 = (目标日数 - 1) mod 12
流日命宫 = (流月命宫 + 日偏移) mod 12
```

### 6.2 流日天干地支

使用五鼠遁从流日天干推算流时天干。

---

## 7. 流时计算

### 7.1 流时命宫

```
流时命宫 = (流日命宫 + timeIndex) mod 12
```

---

## 8. 流耀（Transit Stars）

### 8.1 流耀的定义

流耀是运限盘中动态变化的星曜，使用运限的天干地支重新计算某些星曜的位置。

### 8.2 流耀包含的星曜

| 星曜 | 计算依据 | 算法来源 |
|------|---------|---------|
| 天魁 | 运限天干 | getKuiYueIndex(天干) |
| 天钺 | 运限天干 | getKuiYueIndex(天干) |
| 文昌 | 运限天干（特殊） | getChangQuIndexByHeavenlyStem(天干) |
| 文曲 | 运限天干（特殊） | getChangQuIndexByHeavenlyStem(天干) |
| 禄存 | 运限天干 | getLuYangTuoMaIndex(天干, 地支) |
| 擎羊 | 运限天干 | getLuYangTuoMaIndex(天干, 地支) |
| 陀罗 | 运限天干 | getLuYangTuoMaIndex(天干, 地支) |
| 天马 | 运限地支 | getLuYangTuoMaIndex(天干, 地支) |
| 红鸾 | 运限地支 | getLuanXiIndex(地支) |
| 天喜 | 运限地支 | getLuanXiIndex(地支) |
| 年解 | 运限地支（仅流年） | getNianjieIndex(地支) |

### 8.3 年昌年曲（特殊）

流年/大限盘中的文昌和文曲**不使用出生时辰**定位，而使用运限的天干查表：

| 运限天干 | 年昌 | 年曲 |
|---------|------|------|
| 甲 | 巳(3) | 酉(8) |
| 乙 | 午(4) | 申(7) |
| 丙/戊 | 申(6) | 午(4) |
| 丁/己 | 酉(7) | 巳(3) |
| 庚 | 亥(9) | 卯(1) |
| 辛 | 子(10) | 寅(0) |
| 壬 | 寅(0) | 子(10) |
| 癸 | 卯(1) | 亥(9) |

### 8.4 流耀的命名规范

每个 scope 的流耀使用不同的名称前缀：

| scope | 前缀 | 示例（天魁） | 示例（文昌） |
|-------|------|------------|------------|
| origin | — | 天魁 | 文昌 |
| decadal | 运 | 大魁 | 大昌 |
| yearly | 流 | 年魁 | 年昌 |
| monthly | 月 | 月魁 | 月昌 |
| daily | 日 | 日魁 | 日昌 |
| hourly | 时 | 时魁 | 时昌 |

### 8.5 流耀的类型

| 流耀 | 类型 |
|------|------|
| 天魁/天钺 | `soft` |
| 文昌/文曲 | `soft` |
| 禄存 | `lucun` |
| 擎羊/陀罗 | `tough` |
| 天马 | `tianma` |
| 红鸾/天喜 | `flower` |
| 年解 | `helper` |

## 9. 年解（特殊）

年解 `nianjie` 仅在流年（scope=`yearly`）时出现。

**年解算法**：

| 年支 | 年解位置 |
|------|---------|
| 子→0 | 戌→8 |
| 丑→1 | 酉→7 |
| 寅→2 | 申→6 |
| 卯→3 | 未→5 |
| 辰→4 | 午→4 |
| 巳→5 | 巳→3 |
| 午→6 | 辰→2 |
| 未→7 | 卯→1 |
| 申→8 | 寅→0 |
| 酉→9 | 丑→11 |
| 戌→10 | 子→10 |
| 亥→11 | 亥→9 |

年解位置 = `(戌的宫位索引 - yearBranchIndex) mod 12`

## 10. 运限与四化

每种运限都有自己的四化：

| 运限类型 | 四化依据 | 说明 |
|---------|---------|------|
| 生年 (本命) | 年干 | 生年四化（也称为"天干四化"） |
| 大限 (decadal) | 大限天干 | 运限四化 |
| 流年 (yearly) | 流年天干 | 流年四化 |
| 流月 (monthly) | 流月天干 | 流月四化 |
| 流日 (daily) | 流日天干 | 流日四化 |
| 流时 (hourly) | 流时天干 | 流时四化 |

四化判断使用同一张四化表（见 `06-complete-astrolabe-logic-reference.md` 步骤 12），只是输入的天干不同。

---

## 11. 流年将前十二神

流年盘中，将前十二神和岁前十二神也会随流年变化。

**岁前十二神**：从流年地支位置开始顺时针排列
**将前十二神**：从流年地支的将星位置开始顺时针排列

---

## 12. 运限流耀计算流程

```
输入: 运限日期, 运限时辰
  │
  ├─ 1. 计算运限的天干地支（年干年支）
  ├─ 2. 计算流年命宫 = 运限地支的宫位索引
  ├─ 3. 计算流月命宫、流日命宫、流时命宫
  ├─ 4. 对每个 scope:
  │      ├─ 天魁/天钺 = getKuiYueIndex(scope天干)
  │      ├─ 文昌/文曲 = getChangQuIndexByHeavenlyStem(scope天干)
  │      ├─ 禄存/擎羊/陀罗/天马 = getLuYangTuoMaIndex(scope天干, scope地支)
  │      ├─ 红鸾/天喜 = getLuanXiIndex(scope地支)
  │      └─ [yearly only] 年解 = getNianjieIndex(scope地支)
  │
  └─ 5. 将流耀按 scope 命名规则重命名后放入对应宫位
```
